# gda2
minimal golang dart angular2 web app

####install:

    go get github.com/santiaago/gda2


####build backend:


    > cd $GOPATH/src/github.com/santiaago/gda2
    > go get ./backend

####build frontend:

    >cd app
    >pub get
    >pub build

####run app:

    > cd $GOPATH/src/github.com/santiaago/gda2
    >backend

Open in **Chromium** in `localhost:8080`
